 Book Explorer

A responsive web application for discovering and saving your favorite books using the Open Library API.

## Features

- Search books by title, author, or keyword
- Browse featured books
- Add books to favorites
- Remove books from favorites
- Persistent storage using localStorage

## Technologies

- HTML5
- Tailwind CSS
- JavaScript (ES6 Modules)
- Open Library API

## Project Structure



## How to Run

1. Open index.html in a web browser
2. Search for books or browse featured books
3. Click "Add to Favorites" to save books
4. View saved books in the Favorites page

## API

Uses [Open Library Search API](https://openlibrary.org/dev/docs/api/search)
